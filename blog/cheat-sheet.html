HTML Cheat Sheet

What is HTML?

This blog entry will serve as a cheat sheet for HTML. We have learned a bit about these concepts in Phase 0 of Dev Bootcamp (DBC), but I will try to use this as a reference for some of the basics.

HTML stands for Hyper Text Markup Language, and is a markup language for describing web pages. A markup language is a set of markup tags, and you may be familiar with some of the syntax if you've seen the inner workings of a website. HTML tags are keywords surrounded by angle brackets: 

<tagname>content</tagname>

HTML tags normally come in pairs like 

<p> and </p>

The first tag in a pair is the start tag, the second tag is the end tag. The end tag is written like the start tag, but with a slash before the tag name. Each HTML tag describes different document content, i.e. the head of the document, the body, and other different sections. Let's go through and each one of the required tags to put together a HTML page. Here is a quick example as a visual reference:

Example of HTML syntax:
<!DOCTYPE html>
            <html>
            <head>
            <title>Page Title</title>
            </head>
            <body>
            
            <h1>My First Heading</h1>
            <p>My first paragraph.</p>
            
            </body>
            </html>

Example Explained

First, all documents must start with a type declaration. The <!DOCTYPE html> declaration you see in the example defines the document type to be HTML. Next, the HTML document itself begins with <html> and ends with </html>. The text between <head> and </head> provides meta data (data about the data) for the document. This information is not displayed on the webpage, but it typically defines the document's title, styles, links, scripts, and other meta information. In HTML5 you can omit the head section, but refer to the documentation to read about that. The text between <title> and </title> provides a title for the document; this title defines the title in the browser toolbar, provides a title for the page when it is added to favorites, and displays a title for the page in search engine results. The text between <body> and </body> describes the visible page content - this is where you put your text, links, pictures or any other information. There have been a lot of changes with HTML5, and there are a number of new elements that you can use. Again, refer to the documentation on W3 schools for a full cheat sheet.
